# 1081. Smallest Subsequence of Distinct Characters

**Link:** https://leetcode.com/problems/smallest-subsequence-of-distinct-characters/

**Difficulty:** Medium

---

## Problem Statement

Given a string `s`, return _the lexicographically smallest subsequence of_ `s` _that contains all the distinct characters of_ `s` _exactly once_.

---

## Examples

**Example 1:**

**Input:** s = "bcabc" \
**Output:** "abc"

**Example 2:**

**Input:** s = "cbacdcbc" \
**Output:** "acdb"

---

## Constraints

- `1 <= s.length <= 1000`
- `s` consists of lowercase English letters.
