# 7. Reverse Integer

**Link:** https://leetcode.com/problems/reverse-integer/

**Difficulty:** Medium

---

## Problem Statement

Given a signed 32-bit integer `x`, return `x` _with its digits reversed._ If reversing `x` causes the value to go outside the signed 32-bit integer range <code>[-2<sup>31</sup>, 2<sup>31</sup> - 1]</code>, then return `0`.

**Assume the environment does not allow you to store 64-bit integers (signed or unsigned).**

## Examples

**Example 1:**

**Input:** x = 123 \
**Output:** 321

**Example 2:**

**Input:** x = -123 \
**Output:** -321

**Example 3:**

**Input:** x = 120 \
**Output:** 21

---

## Constraints:

- <code>-2<sup>31</sup> <= x <= 2<sup>31</sup> - 1</code>
