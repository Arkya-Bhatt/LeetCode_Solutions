# 234. Palindrome Linked List

**Link:** https://leetcode.com/problems/palindrome-linked-list/

**Difficulty:** Easy

---

## Problem Statement

Given the `head` of a singly linked list, return `true` _if it is a palindrome or_ `false` _otherwise._

---

## Examples

**Example 1:**

![alt text](pal1linked-list.jpg) \
**Input:** head = [1,2,2,1] \
**Output:** true

**Example 2:**

![alt text](pal2linked-list.jpg) \
**Input:** head = [1,2] \
**Output:** false

---

## Constraints

- The number of nodes in the list is in the range <code>[1, 10<sup>5</sup>]</code>.
- `0 <= Node.val <= 9`
